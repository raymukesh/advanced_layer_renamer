    Advanced Layer Renamer¶

    Professional QGIS plugin for batch renaming map layers with advanced features

    Overview¶

    Advanced Layer Renamer is a powerful QGIS plugin that allows you to rename multiple map layers at once using a variety of transformation options. Whether you need to add prefixes, apply numbering, clean up names, or use complex regex patterns, this plugin has you covered.

    Key Features¶

    • Smart Numbering

      Add sequential numbers with customizable leading zeros (01, 001, 0001, etc.)

    • Find & Replace

      Simple text replacement or powerful regex pattern matching

    • Case Conversion

      UPPERCASE, lowercase, Title Case, or Capitalized

    • Date Stamping

      Automatically add dates in multiple formats (YYYYMMDD, YYYY-MM-DD, DD-MM-YYYY)

    • Templates

      Save and reuse your favorite naming configurations

    • Validation

      Check for naming issues before applying changes

    Quick Example¶

    Transform your layer names in seconds:

    Before After
    my layer (copy) 01_Project_my_layer
    data_2023_final 02_Project_data_2023_final
    untitled layer 03_Project_untitled_layer
